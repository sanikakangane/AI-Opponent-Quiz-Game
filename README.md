# 🚀 QuizClash - AI Opponent Quiz Game ⚡

**QuizClash** is a web-based quiz game where you challenge an AI opponent (“Dev”) across multiple-choice questions. 🧠😈

---

## 🎯 Features

- **Mode Selection**:
  - 🟢 **Free Mode** – playable immediately.
  - 💎 **Premium Mode** – locked, shows upgrade message.
  - 🔥 **Tournament Mode** – locked, shows upgrade message.  

- **Quiz Section**:
  - Timed questions ⏱ (15 seconds per question)  
  - AI answers automatically if you are slow  
  - Score tracking for player and Dev  
  - Pop-up messages for win/lose feedback  

- **Result & Leaderboard**:
  - Result popup shows player and Dev scores  
  - Leaderboard updates with the latest scores  

- **Forum Section 💬**:
  - Users can post messages  
  - Preloaded posts from other users/Dev  
  - User posts appear in green, AI posts in pink  

---

## 💻 Technologies Used

- **HTML** – Page structure, quiz, leaderboard, and forum  
- **CSS** – Styling, layouts, hover effects, popups, and responsive UI  
- **JavaScript** – Quiz logic, AI opponent, timer, leaderboard, forum, and DOM manipulation  

---

## 🕹 How to Play

1. Select a mode (Free, Premium, or Tournament).  
2. Click **Start Quiz**.  
3. Answer the multiple-choice question before the AI does.  
4. Submit your answer or let the AI answer after 5 seconds.  
5. View results in the Result Popup.  
6. Optionally view the **Leaderboard** or post messages in the **Forum**.
